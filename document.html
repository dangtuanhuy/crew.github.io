Introduction

Crew is the library make by Aptech students. It helps you design website interface quickly and conveniently. With Crew's built-in CSS & JavaScript, you can save more time writing code and get more shape for HTML Elements.

Here’s how to quickly Get started with the Crew CDN and a template starter elements.

Quick Start

Looking to quickly add Lumito to your project? Use the Crew CDN as the example below.
- Or you can Download the source files to use.

Copy-paste the stylesheet <link> into your <head> before all other stylesheets to load our CSS.

<link rel="stylesheet" href="https://example.com/crew.css">

Add our JavaScript near the end of your pages, right before the closing tag.

<script src="https://example.com/crew.js"></script>

If you’re at all unsure about the general page structure, keep reading for an example page template.

Starter template

You can use combination with Bootstrap 4 or any other library. That means using an HTML5 doctype and including a viewport meta tag for proper responsive behaviors. Put it all together and your pages should look like this:

<!DOCTYPE html>
<html>
   <head>
       <!-- Required meta tags -->
       <meta charset="utf-8">
       <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

       <!-- Crew CDN-->
       <link rel="stylesheet" href="https://example.com/crew.css">
   </head>
   <body>
      Your code start here
      <!-- Crew JavaScript CDN -->
      <script src="http://animate.luminhthong.com/javascript/animate.js"></script>
   </body>
</html>

That’s all you need for overall page requirements. Read the content below to learn how to use this library.
