GET STARTED

Quickly include Crew's CSS and JS into any project.
Use your favorite package manager, download the source, or use the Crew CDN below.

Crew CDN

You just need to include Crew's compiled CSS and JS to your HTML File.

<!--Crew CDN-->
<link rel="stylesheet" href="https://example.com/crew.min.css">
<script type="text/javascript" src="https://example.com/crew.min.js"></script>
